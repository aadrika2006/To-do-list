# 📝 To-Do List Web App

A simple, modern, and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**. This project helps users organize their daily tasks in a clean and user-friendly interface. It is designed for beginners who want to understand how front-end web development works using core web technologies.

The application allows users to add tasks, remove tasks, and manage their to-do items easily without using any external libraries or frameworks. Its responsive and minimal design makes it suitable for both desktop and mobile screens.

---

## 🚀 Project Overview

The **To-Do List Web App** is a beginner-friendly project that demonstrates the fundamentals of web development. It combines:

- **HTML** for creating the structure of the application
- **CSS** for styling the interface and making it visually appealing
- **JavaScript** for handling user interaction and dynamic task management

This project is useful for learning concepts such as:

- DOM manipulation
- Event handling
- Input validation
- Responsive design
- Interactive UI behavior

## ✨ Features

- Add new tasks to the to-do list
- Delete tasks individually
- Add tasks quickly using the **Enter key**
- Clean and modern user interface

- Easy-to-understand code structure for beginners
- Lightweight application with no external dependencies

## 💡 How the App Works

1. The user types a task into the input field.
2. The task can be added by:
3. Clicking the Add button
4. Pressing the Enter key
5. The entered task appears in the task list dynamically.
6. Each task has a delete/remove option.
7. When the user clicks remove, that task is deleted from the list.

This makes task management simple, fast, and interactive.
